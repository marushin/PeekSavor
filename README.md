# PeekSavor

**PeekSavor**는 로컬 언어 모델(LM Studio 등)을 활용하여 이미지를 분석하고, 이를 바탕으로 Stable Diffusion 및 AI 비디오 생성 도구(Sora, RunwayML, Pika 등)를 위한 프롬프트를 자동으로 생성해주는 강력한 Windows 데스크톱 애플리케이션입니다.

## 🚀 주요 기능

### 🎨 Image Studio (이미지 스튜디오)
- **이미지 분석 및 프롬프트 생성**: 이미지를 드래그 앤 드롭하여 시각적 요소를 분석하고 텍스트 프롬프트로 변환합니다.
- **프롬프트 옵션 상세 활용법**:
  - **기본 모드 (Default)**: Stable Diffusion/ComfyUI 사용자를 위해 **콤마(,)로 구분된 키워드** 위주로 생성합니다.
  - **Detailed**: 빛의 방향, 질감, 구도, 색상 등을 **매우 세밀하게 묘사**합니다. 고품질 이미지 생성을 원할 때 권장합니다.
  - **Natural Language**: "A picture of..."와 같이 **완전한 문장 형태**의 자연스러운 설명을 생성합니다. Midjourney나 일반적인 이미지 설명이 필요할 때 유용합니다.
  - **Korean**: 모든 분석 결과를 **한국어**로 정중하게 출력합니다.
  - **OCR**: 이미지 내의 텍스트를 정확하게 추출합니다. **활성화 시 다른 프롬프트 옵션은 무시**되며 텍스트 추출에만 집중합니다.


### 🎬 Video Director (비디오 디렉터)
- **비디오 프롬프트 생성**: 이미지 분석 결과(Base Prompt)와 사용자의 장면 설명(Scene Description)을 결합하여 완성도 높은 비디오 생성 프롬프트를 작성합니다.
- **활용 팁**:
  - **Base Prompt**: 이미지 스튜디오에서 생성된 프롬프트가 자동으로 연동됩니다. (직접 입력도 가능)
  - **Scene Description**: 카메라 무브먼트(Zoom in, Pans), 인물의 동작, 전체적인 분위기 등을 입력하세요.
  - **결합 출력**: 시각적 요소와 움직임이 조화된 **서술형 문장**으로 최종 프롬프트를 생성합니다.

### 🛠️ 사용자 편의성
- **드래그 앤 드롭**: 파일 탐색기에서 이미지를 바로 끌어와서 열기
- **원클릭 복사**: 생성된 프롬프트를 클립보드로 즉시 복사
- **GitHub 업데이트 알림**: 앱 시작 시 새 버전이 있는지 자동으로 확인하고 알림 표시
- **lmstudio에서 관리하는 모델 리스트 불러오기 및 적용**
- **lmstudio에 로드된 모델 원격 Eject 가능 버튼** !!귀한 비디오 메모리를 반납해야 comfyui가 잘돌아갑니다!


## 📋 요구사항

- **Windows 10/11**
- **LM Studio** (서버로 실행)
  - Vision 기능이 있는 모델 로드 필요 (예: `llava`, `bakllava`, `qwen-vl` 등)
  - 26년 1월 15일 기준으로 개발자는 mistralai/devstral-small-2-2512 24B 모델을 주로 사용합니다.
  - 기본 API 주소: `http://localhost:1234/v1`

## 📦 설치 및 실행 방법

### 방법 1: 실행 파일 사용 (권장)
1. `PeekSavor_v0.0.0.exe` (버전별 상이) 파일을 실행합니다.

앱 실행 후 좌측 사이드바에서 설정을 변경할 수 있습니다.
- **API URL**: 로컬 LLM 서버 주소 (기본값: `http://localhost:1234/v1`)
