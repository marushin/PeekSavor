# PeekSavor

**PeekSavor**는 로컬 언어 모델(LM Studio 등)을 활용하여 이미지를 분석하고, 이를 바탕으로 Stable Diffusion 및 AI 비디오 생성 도구(Sora, RunwayML, Pika 등)를 위한 프롬프트를 자동으로 생성해주는 강력한 Windows 데스크톱 애플리케이션입니다.

## 🚀 주요 기능

### 🎨 Image Studio (이미지 스튜디오)
- **이미지 분석 및 프롬프트 생성**: 이미지를 드래그 앤 드롭하여 시각적 요소를 분석하고 텍스트 프롬프트로 변환합니다.
- **다양한 프롬프트 옵션**:
  - **Detailed**: 매우 자세한 묘사
  - **Natural Language**: 자연어 문장 형태
  - **Korean**: 한국어 묘사 지원
  - **OCR**: 이미지 내 텍스트 추출 (Text Extraction)


### 🎬 Video Director (비디오 디렉터)
- **비디오 프롬프트 생성**: 이미지 분석 결과(Base Prompt)와 사용자의 장면 설명(Scene Description)을 결합하여 완성도 높은 비디오 생성 프롬프트를 작성합니다.
- **AI 비디오 생성기 최적화**: Sora, Runway Gen-2, Pika Labs 등에서 바로 사용할 수 있는 포맷 제공


### 🛠️ 사용자 편의성
- **드래그 앤 드롭**: 파일 탐색기에서 이미지를 바로 끌어와서 열기
- **원클릭 복사**: 생성된 프롬프트를 클립보드로 즉시 복사


## 📋 요구사항

- **Windows 10/11**
- **LM Studio** (서버로 실행)
  - Vision 기능이 있는 모델 로드 필요 (예: `llava`, `bakllava`, `qwen-vl` 등)
  - 기본 API 주소: `http://localhost:1234/v1`

## 📦 설치 및 실행 방법

### 방법 1: 실행 파일 사용 (권장)
1. `PeekSavor_v*.exe` (버전별 상이) 파일을 실행합니다.


## ⚙️ 설정

앱 실행 후 좌측 사이드바에서 설정을 변경할 수 있습니다.
- **API URL**: 로컬 LLM 서버 주소 (기본값: `http://localhost:1234/v1`)
- **Model**: 사용할 비전 모델 선택 (새로고침 가능)
